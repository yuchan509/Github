# Github
Git 사용법

Step1.
git config --global user.email 'my email'
git config --global user.name 'my name'

Step2.
git init

Step3. Change branch 
git branch -M main

Step4. Check status
git status

Step5. Commit
git add *
git commit -m 'My Git File Name'

Step6. Remote
git remote add origin 'My Git URL'

Step7. Pull
git pull --rebase origin main

Step8 Push
git push origin main
